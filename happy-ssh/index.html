# HappySSH

基于Electron+vue3+ts+ssh2实现的ssh连接工具
目前只实现了基础的ssh连接功能

# Github

https://github.com/pumelotea/happy-ssh (opens new window)

# 本地运行

cd client
yarn electron:serve
cd server
yarn run

# 打包编译

修改client/src/background.ts取消// require('./agent')注释

yarn electron:build
