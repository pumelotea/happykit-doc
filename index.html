# 框架介绍

happykit不依赖任何第三方的UI库，开发者可以使用自己喜欢的UI库进行开发。happykit主要关注数据状态的维护，以及核心逻辑的实现； 通过数据适配器的思想进行数据的交互，通过提供类似工厂的方法进行实例创建。

# 技术特性

  • 基于Typescript
  • 基于vue-router
